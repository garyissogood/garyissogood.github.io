hexo Blog頁面與Next主題

發表於 2020-02-10 更新於 2020-02-18 分類於 學習趣 ， 前端 所需閱讀時間 ≈ 4 分鐘

延續 hexo 架設Blog紀錄 後續，繼續添加分類頁面標籤頁面等，並使用Next主題修改的紀錄過程。

新增標籤及分類頁面

hexo部分頁面是要手動生成,使用Git Bash Here 陸續打下以下指令

打完指令會可以到 source 資料夾出現相對應名稱的資料夾,皆有各自的 index.md 檔案

先選tags,打開index.md,輸入 type: “tags”

選categories,打開index.md,輸入 type: “categories”

選about,打開index.md,輸入 type: “about”

顯示滾動百分比

在主題 (Theme) 配置文件_config.yml中,找到scrollpercent: false 修改成 scrollpercent: true 即可呈現滾動百分比

修改頭像

在主題 (Theme) 配置文件_config.yml中,查詢 avatar,修改以下資料

版權宣告

取消底下的由 Hexo 強力驅動 vX.X.X | 主題 — NexT
在主題 (Theme) 配置文件_config.yml中,查詢 powered:與 theme:,將其中屬性 enable: 都改成 false

找到copyright:,可換上自己想要的文字,例如 Gary - 轉載文章請附上作者及原文連結

Note標籤

Note標籤 (Bootstrap Callout) 寫法

要寫在標籤內的內容

[class]有以下幾種,選擇更換 : default | primary | success | info | warning | danger.
[no-icon] : 主題 (Theme) 配置文件有開啟標籤小圖示,可以直接加上直接隱藏

以下各種個顏色模式的標籤 :

default

primary

success

info

warning

danger

在主題 (Theme) 配置文件_config.yml中,查詢 note:裡的style,有四個模式

下一排icons:裡的用 false/true 決定是否開啟標籤內的小圖示

另外其他標籤寫法可以參考 完整的標籤列表
